Javaの特徴

JavaはC言語をベースに開発されたプログラミング言語です。また、Googleが開発に用いる三大言語の1つとしても知られています。

Javaで記述されたプログラムは家電の組み込みやAndroid向けのスマホアプリ、金融機関の基幹システムなどに採用されています。

Javaは実行環境の自由度が高く、一度書いたコードはJVM (Java Virtual Machine) を通してどんなコンピュータでも使えるため、プラットフォームを問わず実行できます。

またJavaは、ソースコードをパソコンなどの機械が理解可能な言語に翻訳してから、処理が実行されるコンパイラ言語です。そのため処理速度が速く安定しており、基幹システムや組み込みでの使用にも耐えるため、重要なシステムを開発する点において大きなメリットといえます。

変数についての説明（宣言など）

まず、変数とは、プログラム内で使用するデータを保持するための、いわば入れ物のようなものです。
この入れ物には、数値や文字列などを保管することができ、保管した値をとりだして参照するなどといったことができます。

他の変数との区別をつけるためにJavaにおいては、変数を宣言する際には、

  • 数値や文字列など、どんな値が格納されるのか
  • どんな名前にするのか
  • 最初に格納する値は何なのか

という点に気を付けなければなりません。

初期化式

初期化式はfor文が実行されるときに最初の一回だけ実行される式で、 条件式で使用する変数の初期化などを主に行います。

条件式

条件式はfor文の繰り返し処理をするための条件のようなものです。
条件式内の条件と合致している間{ }の中のブロック内に記述された処理を上から順に実行し、 条件と合致しなくなったら繰り返し処理を終了します。

変化式

繰り返しが一度実行されブロックの最後に到達したら、最後に変化式が数を変化させます。
変化させるといっても、条件式で使用する変数を増やすための処理です。そのため、変数に対して+ 2や* 2などをこの部分で適用させます。
