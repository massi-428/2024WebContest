Javaの特徴

javaはC言語をベースに開発されたプログラミング言語です。また、Googleが開発に用いる三大言語の1つとしても知られています。 javaで記述されたプログラムは家電の組み込みやAndroid向けのスマホアプリ、金融機関の基幹システムなどに採用されています。
Javaは実行環境の自由度が高く、一度書いたコードはJVMを通してどんなコンピュータでも使えるため、プラットフォームを問わず実行できます。
またjavaは、ソースコードをパソコンなどの機械が理解可能な言語に翻訳してから、処理が実行されるコンパイラ言語です。そのため処理速度が速く安定しており、基幹システムや組み込みでも使用に耐え、重要なシステムを開発する点において大きなメリットといえます。

変数についての説明（宣言など）

まず、変数とはなにかですが、変数とは、プログラム内で使用するデータを保持するためのいわば入れ物のようなものです。
この入れ物には、数値や文字列などを保管することができ、保管した値をとりだして参照するなどといったことができます。 他の変数との区別をつけるためにjava言語においては、変数を宣言する際に以下のような注意点があります。
・数値や文字列など、どんな値が格納されるのか
・どんな名前にするのか
・最初に格納する値は何なのか
変数の宣言
データ型 変数名;
とする必要がある。

int webcon
    webcon = 1

というように変数名(上記の場合はwebcon)を宣言し、そこに値を代入して初期化することで変数を扱うことができます。

また、変数のデータ型にも様々な種類があります。
ここでは基本的に使う頻度が多い「int」、「double」、「String」についての説明を軽くします。

はじめに、intについてですが、intとは整数を格納するための変数です。わかりやすく例でしめすと、

int result = 1

このように記入した場合、「resultは1となる。」となります。整数の値を代入するときはintを使用します。
整数ときたらお次は少数のdoubleです。doubleは少数を格納するための変数です。

double k = 1.23

このように記入した場合、「kは1.23となる。」となります。
そして、Stringについてですが、Stringはほかの2つとはちょっと違い、文字列を代入するためのものです。

String name; name = "Webコン"

HelloWorld(コンソールに出力する系統のコードの説明)

次に、文字を入出力するときに使う関数についてです。
javaの場合、print、printlnという関数を使用します。2つの関数の使い方は以下の通りです。

System.out.print(表示する内容)
System.out.println(表示する内容)

2つの関数の違いについてですが、printは末尾に改行が付きませんが、printlnには末尾に自動的に改行がつくという違いがあります。

四則演算

加算・減算・乗算・除算の四則計算と知っててうれしい+αを紹介します。
加算は+、減算は-、乗算は*、除算は/とそれぞれに算術演算子となるものが存在します。
また、+1や-1を入力するのが面倒くさいなと思ったら、それぞれ++、--という風にすることができます。
そして数字だけでなく文字列も連携することができます。 

 webcon1 = 75;
int webcon2 = 3;

と入力し、その後

 result = webcon1 + webcon2;

と入力をすると結果として

result = 78;

となります。

if文に類するコードの説明

if構文は英語の通り、「もし、??だった場合」という条件分岐のことです。
条件分岐はプログラミングにおけるもっとも大切なことの1つなのでしっかり覚えましょう。
if文の書き方は以下のように書きます。

if(条件式){
    //条件式が真の場合の処理
}
else{
    //条件式が偽の場合の処理
}

パッとみただけではわかりにくいかもしれないので例とともに説明しようと思います。


public class Main {
    public static void main(String[] args){
        int webcon = 10;
        if ( webcon == 10 ){
            System.out.println("webconは10です。");
        }
        else{
            System.out.println("webconは10ではありません。");
        }
    }
}

上の例を実行すると、「webconは10です。」と表示されるはずです。
上の例は、webconに書かれた数を10かそれ以外かを判別し、10の場合は
「webconは10です。」
それ以外だと
「webconは10ではありません。」
と表示されます。

このようにif文は、こちらが指定した条件に合うものと合わないものをわけることができる優れたものです。
プログラミングでできることがとても増えるのでこれをfor構文と一緒に覚えましょう。

for文に類するコードの説明文

for文は指定した回数だけ処理をするときに使われます。for構文は以下のように書きます。

for( 初期化式; 条件式; 変化式){
    繰り返し実行をしたい処理
}

初期化式、条件式、変化式についてよくわからないと思うので、1つずつ解説していきます。
まず、初期化式についてですが、初期化式は for文が実行されるときに最初の一回だけ実行される式で、条件式で使用する変数の初期化などを主に行います。
次に条件式についてですが、これはfor文の繰り返し処理をするための条件のようなものです。
条件式にあうと、{}の中のブロック内に記述された処理を上から順に実行し、条件に合わなくなった場合繰り返し処理を終了し for 文の次へ処理を映します。
繰り返しが一度実行せれブロックの最後に到達したら、最後に変化式が数を変化させます。
変化させるといっても+2や*2などを最後につけたすというものですが。
言葉では難しいと思うので例を下に書いて説明しようと思います。

for (int webcon = 0; webcon < 2; webcon++){
    System.out.println("iwebcon=" + webcon);
}

このように入力した場合以下のような手順で処理が実行されます。

①　初期化式で変数 webcon の宣言と初期化(webcon = 0)
②　条件式を評価 webson < 2 は条件を満たすので繰り返しを実行
③　ブロック内の処理を実行(画面に変数 webcon の値を出力)
④　繰り返しの1回目終了
⑤　変化式で変数 webcon に格納されている値を 1 増加(webcon = 1)
⑥　条件式を評価 webcon < 2 は条件を満たすので繰り返しを実行
⑦　ブロック内の処理を実行(画面に変数 webcon の値を出力)
⑧　繰り返しの2回目終了
⑨　変化式で変数 webcon に格納されている値を 1 増加(webcon = 2)
⑩　条件式を評価 webcon < 2 は条件を満たしていないので繰り返しを終了

結果的に繰り返し処理は2回行われ、それぞれ webcon=0 と webcon=1 を画面に出力します。
if構文と一緒に覚えることでできることがとても増えるので是非覚えてほしいです。(2回目)
