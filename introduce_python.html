はじめに

皆さんPythonに触れたことはありますでしょうか？
初めてプログラミングをする人は大体まずはCかこの言語に手を出すと思います。
ここでは、Pythonの本当に基礎的な部分に触れていこうかなと思います。

Pythonの特徴

コードの記述が短くシンプルな言語で、初心者向けでいろいろなところで利用されています。

主な特徴として、人が書いたプログラムを機械語に翻訳して実行する「インタプリタ言語」であるため、CやJavaと異なりコンパイルを必要としないことが挙げられます。

変数についての説明

変数には様々な役割がありますが、それらの例を以下に示します。

  1. データ保持

    2. 変数は、計算の結果、ユーザー入力、ファイルから読み込んだデータなど、様々な種類のデータを保持します。

  2. 再利用

    3. 一度変数にデータを格納すると、プログラムの異なる部分で何度でもそのデータを使用可能になります。

  3. 可読性の向上

    4. 適切な名前を持つ変数は、プログラムコードを読みやすくします。

  4. 変更の管理

    5. プログラム内の複数の場所で使用されるデータを変更する必要があるとき、そのデータを変数に格納しておけば、 一か所で変更するだけでプログラム全体で変更が反映されます。

  5. 動的な操作

    6. 変数の値は、プログラムが実行されている間は動的に変更されます。

変数の設定

x=5

Pythonにおいて変数は=で作成することができる。左側に変数名を、右側に格納したい値を置きます。

上記した変数は変数名がxで、格納された値は5であるので、この変数を読み込むと5が読み込まれます。

一度定義した変数には新たな値を代入することができます。上記の変数xに新たな値を代入する場合には 

x=10

このようにすれば実現できます。こうすることにより、最新の値である10が変数に格納されることになります。
Pythonでは、一つの変数から別の変数へ値を代入することができるのです。

コンソールに出力する系統のコードの説明

print("Hello World")

上記したプログラムを実行すると、コンソールにHello Worldと出力することができます。

Pythonにおいてコンソールに出力する際は、基本的にprintを用いて出力します。
printは 

print("出力したいもの")

このようにprintの後に括弧とダブルクオーテーションを使い出力したいものを囲むことによって扱うことができます。

四則演算

Pythonの四則演算のプログラムは、 

("〇演算記号〇="演算記号="演算記号(〇演算記号〇))

であらわすことができます。

上記したプログラムの〇の部分には任意の数字が入り、演算記号の部分には+、-、*、/のいずれかが当てはまります。 

str(4+3)

このコードを実行すると、コンソールには下記のように表示されます。

4+3=7

以上が四則演算のプログラムコードの説明です。

if文に類するコードの説明

Pythonのif文は、条件分岐を実現するために使われます。指定した条件がTrue(真)であれば、 そのブロック内のコードを実行し、False(偽)であれば実行しないという処理を実装することができます。 

wake_up_time=7
if wake_up_time==7:
    print("朝ご飯を食べる")

上記したプログラムコードがif文のサンプルコードです。

上記したプログラムは変数wake_up_timeを7として格納し、if文によって前述した変数が7である時に「朝ご飯を食べる」とコンソールに出力されるようになっています。

for文に類するコードの説明

for文は、リストやタプル、辞書、セットなどの要素を順番に取り出して処理を行う制御構文です。

for文は、処理の回数が決まっているループ処理でカウンターの宣言と初期化、条件式、カウンターの更新式の3つの要素から構成されています。 

for w in words:
    print(w)

この関数を実行した場合、変数wに格納されているものが順番にコンソールに出力されていきます。
