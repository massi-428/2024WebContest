Rustの特徴

Rustは、メモリ効率がよく、開発者向けのプログラミング言語です。
データ型のサイズ指定などができ、メモリを最低限に抑えて作業を行うことができます。
C,C++の派生言語であり、C言語を既に習得している人なら割と楽に習得が可能です。

早速実践

~宣言について~

宣言にはletを使います。
Rustでは、変数宣言にはmutをつけます。mut無しでも勝手に変数宣言として扱ってくれますが、見やすいコードを書くためにmutをつけましょう。

let mut x;

また、変数型のほかに定数型が存在します。
整数型では、符号付きか符号なしかを選べ、明示的なサイズを持ちます。
また、可変,不変の設定も行うことができ、不変の設定を行うと、プログラムが予期しない動作を起こしたときにエラーを吐いてくれます。

~テキストを入出力するための関数~

基本的にはC言語と大差ありません。
printfがprint!になっただけです。
かっこの中身もほぼ変わっていませんし、同じと言っても過言ではないでしょう。

print!("xの値は{}です。",x);

~四則演算~

こちらもC言語と大差ありません。

x=1+2;
y=10%3;

あらかじめ用意していた変数に+,-,*,/,%などを使って導き出した計算結果を代入します。
ただし、変数を事前に用意していないとしっかりエラーを吐きます。

~if文に類するコード~

if a in b{print!("aとbは同じ値です。");}
else if a!=c{print!("aとcは同じではありません。");}
else{print!("えるす。");}

こちらはC言語とは打って変わって、どちらかといえばpython寄りのコードとなっています。
基本的な動作としてはif 条件{条件に合致した時の動作};と、他の言語と同じような概要です。

~ループに類するコード~

for,whileは存在します。 

for a in b{print!("aはbより小さいです。");}
while a==0{print!("aは0です。");}

使い方は変わりませんが、条件文はちょっとpython似です。

また、Rustにはloopという関数が存在します。
最も簡単なループ文(終了条件もループ条件も書かれていないコード)を書くとき、

loop{print!("るーぷ")};

これだけでループ文が完成します。
簡単なコードがあって便利ですね。
